# 📊 Superstore SQL Analysis Project

## 📌 Project Overview
This project focuses on analyzing a Sales (Superstore) dataset using SQL.  
It includes data exploration, filtering, sorting, aggregation, grouping, subqueries, and window functions to generate meaningful business insights.

---

## 🛠️ Tools Used
- SQL Server / MySQL
- Git & GitHub

---

## 📁 Dataset
- Table Name: Sales

---

## 🔍 Project Highlights
- Data exploration using SELECT queries  
- Filtering data using WHERE clause  
- Sorting results using ORDER BY  
- Aggregations using GROUP BY and HAVING  
- Subqueries for advanced filtering  
- Window functions (RANK, DENSE_RANK, ROW_NUMBER)  
- Business insights like top sales, profit analysis, and regional performance  

---

## 📊 Key Insights
- Identified top-performing products and customers  
- Analyzed sales distribution across regions and categories  
- Found highest profit and second highest sales records  
- Performed ranking-based analysis on sales data
